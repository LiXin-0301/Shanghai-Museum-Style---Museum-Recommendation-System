1.请将springboot和backage项目中的application.yml文件中的数据库信息和图片存取信息按照电脑具体情况进行设置。
2.请确认相关环境配置完全，不完全包括IntelliJ IDEA版本，jdk，node.js，vue,element,chart等一系列需要用到的插件，可以参考报告->2.2系统软硬件平台 中详情。
3.功能完整需要同时打开springboot和backage项目，如仅打开springboot项目会无法看见图片和可能进行的管理员操作。
4.由于图片在服务器端储存，因此代码附带的图片url无效，需要根据您的个人服务器情况重新利用backage(后台管理系统)导入，在导入时需要仔细检查您的文件是否导入成功，博物馆主图照片需要在m_name中加上”主图“二字便于系统辨识。
5.细节问题可以上网查询获得帮助。
